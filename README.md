# Password Manager (Django)

A simple password manager built with Django.

## Features

- User login and logout
- Add and view encrypted passwords
- Reveal decrypted passwords with master password

## How to run

1. `python -m venv venv`
2. `venv\Scripts\activate`
3. `python -m pip install django`
4. `python manage.py migrate`
5. `python manage.py runserver`

# Password Manager (Django)

A secure, modern password manager built with Django to manage credentials with AES-style encryption.

## Features

- **Secure Authentication:** Custom Django Auth for a professional user experience.
- **Data Security:** AES-style encryption ensures no plain-text passwords in the database.
- **Responsive UI:** Modern design using Bootstrap 5 with a glassmorphism aesthetic.
- **Session Management:** Secure handling of master passwords.

## Tech Stack

- **Backend:** Django (Python)
- **Frontend:** Bootstrap 5 (Responsive Design)
- **Security:** Python Cryptography (AES)
- **Tools:** Git & GitHub
- **Database:** SQLite
